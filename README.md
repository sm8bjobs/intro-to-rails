# Intro to Rails

## Overview
This is a practice project for learning Ruby on Rails.

## Features
- Basic CRUD functionality.
- User authentication.

## Technologies Used
- Ruby
- Rails

## Usage
1. Clone the repository.
2. Install dependencies: `bundle install`
3. Start the server: `rails server`
